## Hi there 👋

This repository hosts my personal **editable portfolio** website.

👉 **[View Portfolio →](https://lamlam44.github.io/Lamlam44/)**

### ✨ Features
- **Editable intro** — click *Edit* to update your name, title, bio and contact info
- **Avatar** — upload a photo or paste an image URL
- **Skills** — add or remove skill tags in one click
- **Projects** — add/remove project cards with links, tags and descriptions
- **Auto-save** — all changes are stored in your browser's localStorage

### 🚀 Usage
1. Open `index.html` in any browser (or deploy via GitHub Pages).
2. Click the **✏️ Edit** button in the top-right corner.
3. Click any text to change it, manage skills and projects, then press **💾 Save**.
